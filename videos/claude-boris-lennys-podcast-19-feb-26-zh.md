[`11:50`](https://youtu.be/We7BZVKbCVw?t=710) 实际上在很早的时候，在我发布它之前，Ben Mann 就推动我做一个 DAU 图表，我当时想，你知道，可能有点早，也许，你知道，我们现在真的应该做吗？他说，"是的。"然后图表立刻就垂直起飞了。呃，然后在二月，我们在外部发布了它。实际上，有些人不记得的是，Claude Code 在发布时并不是立即大获成功的。它获得了一批用户。有很多早期采用者立刻理解了它，但实际上花了好几个月才让每个人都真正理解这个东西是什么。再说一次，它太不同了。而且当我回想起来时，Claude Code 之所以有效，部分原因在于"潜在需求"这个理念：我们把工具带到人们所在的地方，让现有的工作流程稍微容易一些，但同时因为它是在终端里，有点令人惊讶，有点陌生。所以你必须保持开放的心态，你必须学会使用它。当然现在，你知道，Claude Code 在 iOS 和 Android 的 Claude app 中可用。在桌面应用中可用。在网站上可用。在 IDE 扩展、Slack 和 GitHub 中也可用。你知道，所有这些工程师所在的地方，它都更熟悉一些，但那不是起点。
