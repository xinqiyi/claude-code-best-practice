## 提示

| ID | 内容 |

| default-permission-mode-config | 使用 /config 更改默认权限模式（包括计划模式） |
| git-worktrees | 使用 git worktrees 并行运行多个 Claude 会话。 |
| color-when-multi-clauding | 同时运行多个 Claude 会话？使用 /color 和 /rename 来一目了然地区分它们。 |
| memory-command | 使用 /memory 查看和管理 Claude 记忆 |

| colorterm-truecolor | 尝试设置环境变量 COLORTERM=truecolor 以获得更丰富的颜色 |
| powershell-tool-env | 设置 CLAUDE_CODE_USE_POWERSHELL_TOOL=1 以启用 PowerShell tool（预览版） |
| status-line | 使用 /statusline 设置自定义状态行，将显示在输入框下方 |
| prompt-queue | 在 Claude 工作时，按 Enter 键排队发送更多消息。 |
| enter-to-steer-in-relatime | 在 Claude 工作时发送消息，实时引导 Claude |
| todo-list | 在处理复杂任务时，让 Claude 创建待办事项列表以跟踪进度并保持在正轨上 |
| ide-upsell-external-terminal | 将 Claude 连接到你的 IDE · /ide |
| install-github-app | 运行 /install-github-app 即可从 GitHub issue 和 PR 中直接 @ 提及 Claude |
| install-slack-app | 运行 /install-slack-app 在 Slack 中使用 Claude |

| continue | 运行 claude --continue 或 claude --resume 来恢复对话 |
