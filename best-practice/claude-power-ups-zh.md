# Power-ups 最佳实践

![Last Updated](https://img.shields.io/badge/Last_Updated-Apr%2002%2C%202026-white?style=flat&labelColor=555)

交互式课程，通过动画演示教授 Claude Code 功能。每个 power-up 介绍一项 Claude Code 能做但大多数人不知道的功能。于 v2.1.90 引入。

## 使用方法

```bash
claude
/powerup
```

---

## Power-ups（共 10 个）

<p align="center">
  <img src="assets/claude-power-ups/powerup-menu.png" alt="Power-ups 菜单，展示 10 个课程" width="700">
</p>

| # | Power-up | 主题 |
|---|----------|--------|
| 1 | 与代码库对话 | `@` 文件、行引用 |
| 2 | 用模式掌控 | `shift+tab`、plan、auto |
| 3 | 撤销任何操作 | `/rewind`、`Esc-Esc` |
| 4 | 后台运行 | tasks、`/tasks` |
| 5 | 教 Claude 你的规则 | `CLAUDE.md`、`/memory` |
| 6 | 用工具扩展 | MCP、`/mcp` |
| 7 | 自动化你的工作流 | skills、hooks |
| 8 | 分身多任务 | subagents、`/agents` |
| 9 | 随处编码 | `/remote-control`、`/teleport` |
| 10 | 切换模型 | `/model`、`/effort` |
